# CodeAlpha Phishing Awareness Training

## Project Overview
This project was completed as part of the CodeAlpha Cyber Security Internship.

The aim of this project is to educate users about phishing attacks, social engineering techniques, and cybersecurity best practices for staying safe online.

---

## Presentation Topics
The presentation covers:

- Introduction to phishing
- Types of phishing attacks
- Social engineering tactics
- Warning signs of phishing emails
- Real-life phishing examples
- Cybersecurity protection tips
- Response steps after phishing attacks
- Interactive quiz session
